# Ticker_fetching_app

Simple GUI app to fetch daily OHLC data for a given stock ticker and date using yfinance.

## Requirements

- Python 3.8+
- pip packages:
  - yfinance
  - pandas
- tkinter (usually included with Python on macOS)

Install requirements:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install yfinance pandas
